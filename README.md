# VPN Service Web Interface

Данный проект представляет собой современный веб-интерфейс для управления VPN-сервисом, разработанный в рамках учебной практики. Система обеспечивает взаимодействие пользователя с услугами защищенного соединения, выбор тарифных планов и автоматизированное управление конфигурациями доступа.

## 🚀 Актуальность
В условиях современной цифровой среды вопрос обеспечения приватности и безопасности передачи данных стоит максимально остро. Проект решает следующие задачи:
* **Защита данных:** обеспечение безопасности в публичных сетях и предотвращение утечек личной информации.
* **Доступность:** обеспечение стабильного доступа к сетевым ресурсам вне зависимости от географических ограничений.
* **Цифровая гигиена:** сокрытие сетевой активности от сторонних наблюдателей и провайдеров.

## 🛠 Технологический стек
Проект реализован с использованием актуального стека технологий:

| Слой | Технология |
| :--- | :--- |
| **Frontend** | React.js, JavaScript (ES6+) |
| **Build Tool** | Vite |
| **Styling** | Custom CSS (Standard modules) |
| **Automation** | GNU Make (Makefile) |
| **Protocols** | WireGuard / OpenVPN |

## ⚙️ Автоматизация (Makefile)
Для упрощения развертывания и управления проектом настроен `Makefile`. Основные команды:
* `make install` — установка всех зависимостей.
* `make dev` — запуск сервера разработки (Vite).
* `make build` — создание релизной сборки.
* `make push m="message"` — быстрый коммит и пуш в репозиторий.
* `make stop` — экстренное завершение зависших Node-процессов.
